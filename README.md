# Credit Card Financial & Customer Analytics Dashboard

## Overview
This project is a professional Business Intelligence and Data Analytics solution developed using Power BI, SQL, Power Query, and DAX. The dashboard provides comprehensive insights into customer behavior, transaction trends, revenue generation, and financial performance through interactive visualizations and KPI-driven reporting.

The project focuses on transforming raw financial and transactional data into meaningful business insights using data cleaning, transformation, modeling, and advanced analytical techniques.

---

## Objectives
- Analyze customer transaction behavior
- Monitor revenue and financial KPIs
- Track weekly and quarterly business performance
- Identify high-value customer segments
- Generate actionable business insights through data visualization
